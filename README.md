# BioIntelForge

Automated scientific literature discovery and summarization system built using n8n.

## Overview

BioIntelForge monitors scientific literature sources such as PubMed and arXiv, filters relevant papers, performs AI-assisted analysis, and generates structured summaries.

## Features

- PubMed integration
- arXiv integration
- Automated relevance scoring
- AI-powered paper analysis
- Structured JSON extraction
- Automated email reporting

## Technologies

- n8n
- Groq API
- PubMed API
- arXiv RSS
- Workflow Automation

## Workflow

PubMed + arXiv
→ Normalization
→ Deduplication
→ Relevance Filtering
→ AI Analysis
→ Structured Summary
→ Email Report

## Achievement

Awarded 2nd Place in a college-wide n8n automation competition.
